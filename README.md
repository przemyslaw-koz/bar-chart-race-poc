# Bar Chart Race POC

Proof of concept for a **bar chart race** animation built with Remotion (React + TypeScript).

The target pipeline architecture (orchestrator, AWS, worker) is documented in [`docs/`](docs/).

## Remotion app

All POC code and run instructions live in **[`app/`](app/)**:

- preview in Remotion Studio (`npm run dev`)
- MP4 export (`npx remotion render` or Docker)
